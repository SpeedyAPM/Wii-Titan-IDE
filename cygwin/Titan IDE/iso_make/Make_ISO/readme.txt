С помощью этого пакета вы можете создавать образы для модуля DreamShell sdiso.
Положите все файлы с диска в директорию cd и кликните 2 раза по файлу Create_{type}_ISO.bat

English:
With this package you can create images for the module DreamShell sdiso.
Put all files from cd to the directory and click 2 times on the file Create_{type}_ISO.bat
